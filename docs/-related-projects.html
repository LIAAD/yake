Related Projects

List of relevant projects using YAKE:

  • SparkNLP by John Snow Labs. Open source framework with full Python, Scala, and Java Support. Check their documentation, demo and google colab. A video on how to use spark nlp with yake can also be found here:

  • pke - Python Keyphrase Extraction module. Ii is an open source python-based keyphrase extraction toolkit. It provides an end-to-end keyphrase extraction pipeline in which each component can be easily modified or extended to develop new models. Credits to Florian Boudin.

  • Textacy - NLP, before and after spaCy. Python library for performing a variety of natural language processing (NLP) tasks, built on the high-performance spaCy library. among other features it supports keyword extration using YAKE.

  • General Index by Archive.org. A catalogue of 19 billions of YAKE keywords extracted from 107 million papers. An article about the General Index project can also be found in Nature.

  • Annif is a tool for automated subject indexing and classification. Multi-algorithm automated subject indexing tool for libraries, archives and museums. This repository is used for developing a production version of the system, based on ideas from the initial prototype. Official website.

  • Portulan Clarin - Provides Sservices and data for researchers, innovators, students and language professionals Research Infrastructure for the Science and Technology of Language, belonging to the Portuguese National Roadmap of Research Infrastructures of Strategic Relevance, and part of the international research infrastructure CLARIN ERIC. It includes a demo of YAKE! among many other language technologies. Official website.

  • Tell me stories is a research project that allows users to automatically build news narratives based on news preserved by the Portuguese Web Archive.
